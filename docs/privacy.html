Política de Privacidad

Última actualización: 18 de junio de 2025

Tu privacidad es importante para nosotros. Esta política explica cómo la aplicación Calypso gestiona la información del usuario.

1. Datos que recogemos

Calypso no recopila ni almacena datos personales. La única información registrada es el tiempo de uso de funciones como streaming o grabación, con fines exclusivamente analíticos internos.

2. Permisos solicitados

Calypso requiere acceso a:

  • Cámara: para grabar o transmitir video.
  • Micrófono: para capturar audio durante las transmisiones o grabaciones.

Estos permisos se solicitan únicamente cuando son necesarios y no se usan para recopilar información sin consentimiento.

3. Almacenamiento y transferencia de datos

No se envía información a servidores externos ni se comparten datos con terceros. Toda la información se guarda localmente en el dispositivo.

4. Análisis y estadísticas

Se registran métricas de uso como la duración de las sesiones de streaming o grabación. Esta información no está vinculada a ninguna identidad personal.

5. Cambios en esta política

Podemos actualizar esta política ocasionalmente. Notificaremos los cambios importantes en la aplicación o a través de nuestra página oficial.
